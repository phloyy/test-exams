# ИКТ Экзамен - Интерактивный Тест

Интерактивный тест для подготовки к экзамену по ИКТ в экономике.

## Возможности

- **Режим тренировки**: Мгновенная проверка с AI-пояснениями
- **Режим экзамена**: Результаты в конце теста
- **Прогресс**: Отслеживание правильных/неправильных ответов
- **213 вопросов** с вариантами ответов

## Деплой на Vercel

1. Форкни этот репозиторий
2. Зайди на [vercel.com](https://vercel.com)
3. Импортируй проект
4. Vercel автоматически задеплоит приложение

## Локальная разработка

```bash
npm install
npm run dev
```

Открой http://localhost:5173
